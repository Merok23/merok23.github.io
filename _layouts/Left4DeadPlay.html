Instructions:

To start playing the game, follow these steps:

  1. Download the game installer by clicking the download button above.
  2. Save the downloaded file (installer.sh) to a location of your choice.
  3. Open a terminal and navigate to the location where you saved the installer.
  4. Execute the installer by running the command: bash installer.sh

The installer will automatically download and install all the necessary dependencies and the game itself. Once the installation is complete, you will find two additional .sh files on your desktop.

If you see both files on your desktop, it indicates that the game has been successfully installed, and you are now ready to play.

For more detailed instructions on how to play the game, whether locally or online, please refer to the user manual.
